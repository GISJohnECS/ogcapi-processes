|*Requirement {counter:req-id}* |/req/core/job-result-success +

A successful execution of the operation SHALL be reported as a
response with a HTTP status code 200.
The content of that response SHALL be based upon the OpenAPI
3.0 schema https://raw.githubusercontent.com/opengeospatial/ogcapi-processes/master/core/openapi/schemas/result.yaml[result.yaml].
